Timestamp Microservice

Pass any date in your API call and get back JSON with unix and natural date formatting.

Examples:

https://thekholm80-timestamp.herokuapp.com/December 15, 2015

https://thekholm80-timestamp.herokuapp.com/December%2015,%202015

https://thekholm80-timestamp.herokuapp.com/1450162800

Returns:

{ 'unix': '1450137600', 'natural': 'December 15, 2015'}
